# UVa 10170 - The Hotel with Infinite Rooms

## 題目說明
給定兩個整數 `S` 和 `D`：
- 第一天有 S 個人入住
- 第二天有 S+1 個人入住
- 第三天有 S+2 個人入住
- 依此類推

每天的人數會累加，直到總人數 >= D  
請輸出「第幾天」會達到或超過 D。

---

## 解題思路
1. 讀入 S 和 D（可能有多筆測資）
2. 設目前天數的人數為 `cur = S`
3. 累加總人數 `sum += cur`
4. 每一天人數增加：`cur++`
5. 重複直到 `sum >= D`
6. 輸出當下的 `cur`（即當天人數）

---

## 關鍵概念
- 等差數列累加
- 不需要真的記「第幾天」，只要追蹤目前人數即可
