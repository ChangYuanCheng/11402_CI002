# UVa 490 - Rotating Sentences

## 題目說明
輸入多行字串，將所有字串「旋轉 90 度」輸出。

也就是：
- 原本是「一行一行」
- 輸出變成「一列一列」

並且所有字元要以最長字串為準對齊。

---

## 解題思路
1. 讀入所有輸入行，直到 EOF
2. 找出最長字串長度 `maxLen`
3. 建立輸出規則：
   - 第 i 欄 → 輸出每一行的第 i 個字元
4. 若該行沒有第 i 個字元：
   - 輸出空白
5. 逐欄輸出完成旋轉

---

## 關鍵概念
- 「行轉列」= matrix 轉置概念
- 不足長度用空白補齊
- 注意輸出順序是「由左到右欄掃描」

---

## 時間複雜度
- Time: O(n × m)
- Space: O(n × m)
