# UVa 100 - The 3n + 1 Problem

##  題目說明
給定兩個整數 `i` 和 `j`，需要找出區間內所有數字的最大循環長度（cycle length）。

規則如下：
- 若 n 為偶數 → n = n / 2
- 若 n 為奇數 → n = 3n + 1
- 重複直到 n = 1
- 計算總步數（包含起始值與1）

---

##  解題思路
1. 讀入 `i` 和 `j`
2. 先記錄原本的 `i, j`（因為輸出要保持原順序）
3. 若 `i > j`，先交換，方便計算區間
4. 對區間 `[i, j]` 每個數字：
   - 計算其 cycle length
5. 找出最大值
6. 輸出：原本的 `i j` 以及最大 cycle length

---

##  注意事項
- 輸出時要維持「原輸入順序」
- 輸入可能有多筆測資（使用 `while(cin >> i >> j)`）
- cycle length 要包含起始數字本身

---

##  時間複雜度
- Time: O(n × cycle length)
- Space: O(1)
