# UVa 299 - Train Swapping

## 題目說明
給定一列火車車廂的排列順序，目標是將其排序成由小到大。

每次只能交換「相鄰」兩個車廂，請計算最少需要幾次交換。

---

## 解題思路
這題本質是計算「反序對（inversion）」數量。

做法：
1. 讀入車廂數 n
2. 讀入排列
3. 使用 Bubble Sort（或雙層迴圈）：
   - 每當 a[i] > a[j]（i < j），就交換
   - 並計數 +1
4. 最後的交換次數就是答案

---

## 關鍵概念
- 每一次交換 = 修正一個逆序
- Bubble Sort 的交換次數 = 反序對數量
