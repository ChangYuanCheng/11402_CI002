# UVa 11150 - Cola

## 題目說明
給定 n 瓶可樂，每喝完一瓶會得到一個空瓶。

規則：
- 3 個空瓶可以換 1 瓶可樂
- 可以「借 1 瓶」來多換一次（但最後要還）

請問最多可以喝幾瓶可樂。

---

## 解題思路
1. 初始喝掉 n 瓶 → ans = n，空瓶 = n
2. 當空瓶數 >= 3：
   - 換新的可樂：empty / 3
   - 累加到 ans
   - 更新空瓶：empty = empty / 3 + empty % 3
3. 最後如果空瓶剛好剩 2：
   - 可以借 1 瓶 → 多喝 1 瓶

---

## 關鍵概念
- 空瓶循環利用
- 最後剩 2 瓶可以「借 1 瓶」再換一次
