# UVa 10008 - What's Cryptanalysis?

## 題目說明
給定 n 行文字，統計每個英文字母（A~Z）出現的次數（不分大小寫）。

最後依規則輸出：
1. 依出現次數「由大到小」排序
2. 若次數相同，依字母「由小到大」排序

---

## 解題思路
1. 讀入整數 n
2. 用陣列記錄 26 個字母的出現次數
3. 對每一行字串：
   - 逐字檢查
   - 若是字母 → 轉成大寫
   - 對應到陣列 index（A=0, B=1...）
   - 次數 +1
4. 將結果存成 (字母, 次數) 的結構
5. 排序：
   - 先比次數（大到小）
   - 次數相同再比字母（小到大）
6. 只輸出次數 > 0 的字母
