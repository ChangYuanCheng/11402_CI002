# UVa 10035 - Primary Arithmetic

## 題目說明
給定兩個非負整數，模擬十進位加法，計算在加法過程中產生了幾次「進位」（carry）。

輸入多筆資料，每筆為兩個整數，當輸入為 `0 0` 時結束。

---

## 解題思路
1. 讀入兩個整數（以字串形式處理較方便）
2. 從最低位（右邊）開始逐位相加
3. 設一個變數 `carry` 紀錄是否有進位（初始為0）
4. 每一位計算：
   - sum = 當前位數字1 + 當前位數字2 + carry
   - 若 sum >= 10，表示產生進位，carry = 1，並計數+1
   - 否則 carry = 0
5. 持續到兩個數字都處理完
6. 根據進位次數輸出結果

---

## 輸出規則
- 若沒有進位 → `No carry operation.`
- 若有1次 → `1 carry operation.`
- 若超過1次 → `x carry operations.`

---

## 注意事項
- 長度不同時，短的數字前面視為補0
- 必須從最低位開始處理
- 輸入為 `0 0` 時停止，不需輸出

---

## 時間複雜度
- Time: O(n)
- Space: O(1)
