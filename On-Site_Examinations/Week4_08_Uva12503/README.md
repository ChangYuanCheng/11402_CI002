# UVa 12503 - Robot Instructions

## 題目說明
有一台機器人站在位置 0，接下來會收到多個指令：

- `LEFT`  → 往左移動（-1）
- `RIGHT` → 往右移動（+1）
- `SAME AS i` → 執行第 i 個指令（i 從 1 開始）

請輸出最後機器人的位置。

---

## 解題思路
1. 讀入測資數 T
2. 對每一筆：
   - 讀入指令數 n
   - 用一個陣列記錄每個指令的結果（+1 或 -1）
3. 每讀一個指令：
   - LEFT → 記為 -1
   - RIGHT → 記為 +1
   - SAME AS i → 直接取第 i 個指令的值
4. 每次累加到位置 position
5. 最後輸出 position

---

## 關鍵概念
- 用陣列「記住之前指令的結果」
- SAME AS 本質是「複製之前的動作」
