# UVa 272 - TEX Quotes

## 題目說明
將輸入文字中的雙引號進行轉換：
- 第一個出現的 " → ` `` `
- 第二個出現的 " → ` '' `
依序交替轉換，模擬 TeX 排版系統的引號格式。

---

## 解題思路
1. 逐字讀取整行輸入字串
2. 使用一個變數 `flag` 判斷目前是第幾次遇到 `"`
   - 偶數次：輸出 ````（左引號）
   - 奇數次：輸出 `''`（右引號）
3. 若不是 `"`，直接原樣輸出
4. 重複處理直到輸入結束

---

## 注意事項
- 每一行都要獨立處理
- 引號是交替出現，不是固定替換
- 使用 getline 讀整行避免空白問題

---

## 時間複雜度
- Time: O(n)
- Space: O(1)
